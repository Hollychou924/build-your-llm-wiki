# article-archivist Beta 发布口径

## 一句话介绍
`article-archivist` 是一个个人知识库 starter kit：不是帮你“存链接”，而是帮你把公众号、博客、长文网页逐步编译成一个可检索、可升级、可治理的本地知识系统。

## 当前已经能做什么
- 初始化本地知识库骨架
- 统一 ingest 公众号 / 官方博客 / 普通长文网页
- 自动写 raw / memory / 日记 / 总日志
- 自动做六目录检查
- 自动生成知识层初稿
- 对微信文章优先支持 SkillHub + `wechat-article-extractor-skill`
- 支持 workspace 级去重隔离
- 支持默认 beta 回归测试

## 适合谁现在就试
- OpenClaw / Claude Code 用户
- 想搭本地知识库的人
- 愿意接受 Beta、并反馈问题的内部用户 / 熟人

## 不适合谁现在直接用
- 对“一次安装就完全无脑稳定”有强预期的人
- 还不想碰命令行的人
- 对本地知识库结构完全没兴趣、只想临时存链接的人

## Beta 的真实状态
- 主链已跑通
- 默认三类样本回归已通过：公众号 / 官方 blog / 普通长文
- 现在的重点不是“能不能用”，而是继续磨平最后一层粗糙感

## 当前建议的试用方式
1. 先让 1-3 个熟人试装
2. 每人至少喂 3 类链接各 1 篇
3. 回收他们第一次安装、第一次 ingest、第一次看草稿时的卡点
4. 根据反馈继续修说明、抽取器和知识页语言

## 推荐给外部时怎么说
可以说：
- 这是一个可用的 Beta
- 已经适合真实试装和小范围使用
- 现在最需要的是早期试用反馈，而不是等到“完美版”再发布
